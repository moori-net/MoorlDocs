Dieses Plugin erfasst Seitenaufrufe auf Produktseiten und Bestellungen. Auf Grundlage der gesammelten Daten werden automatisch Cross-Selling-Listen erstellt.

## Funktionsweise

Das Plugin analysiert, welche Produkte häufig gemeinsam angesehen oder zusammen gekauft werden. Auf Basis dieser Daten werden automatisch passende Produktbeziehungen erstellt und als Cross-Selling am Produkt hinterlegt.

So entstehen ohne manuellen Pflegeaufwand passende Produktempfehlungen wie:

- Häufig zusammen angesehen
- Häufig zusammen gekauft
- Zuletzt angesehen
