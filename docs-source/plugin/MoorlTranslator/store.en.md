## More Efficiency for Multilingual Shops

Automatic translation helps you provide content in multiple languages more quickly. Recurring translations can be handled efficiently using a lookup table or dictionary, reducing the load on the connected translation service.

## Flexible Configuration

The plugin gives you full control over which content is translated automatically:

- Freely configurable entities
- Freely configurable CMS elements
- Configurable source and target languages
- Easy management directly in the Shopware admin

## Ideal for

- Multilingual online shops
- International product catalogs
- Shops with a high translation maintenance workload
- Installations with additional plugins from other vendors

## Technical Foundation

- Developed for Shopware 6.7
- Integrated into the Shopware admin
- Compatible with extensions from other vendors
